Graph to Dungeon plugin
Procedural dungeon-level generator in a plane, utilizing an undirected graph.

This project consists of two plugins:
  1. The Graph to Dungeon plugin was created from scratch and uses the second plugin.
  2. The Generic Graph plugin is a slightly adjusted version of https://github.com/jinyuliao/GenericGraph.

Unreal Engine 5.3.2 setup
This setup documentation was tested on Windows 11.

1.  Install Unreal Engine 5.3.2 - https://www.unrealengine.com/en-US/download
2.  (Optional) Install Visual Studio 2022 - https://visualstudio.microsoft.com/downloads/
3.  Clone or unzip into your Plugins file inside your UE5 project; if not present, create one.
4.  Launch your project, click yes to rebuild, and wait until your project boots. Alternatively build with VS.
5.  Open Graph to Dungeon window under Widows->Graph to Dungeon.
6.  Create Properties asset or use prepared properties, to do so enable view of plugin content Settings->Show Plugin Content.
7.  Generate New Dungeon.
